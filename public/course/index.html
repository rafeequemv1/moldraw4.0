What is MolDraw?

MolDraw is a powerful, web-based molecular structure editor that allows you to:

  • Draw and edit chemical structures in 2D
  • Visualize molecules in 3D in real-time
  • Search for molecules by name using PubChem
  • Export structures in multiple formats (PNG, SDF, XYZ, X3D, OBJ)
  • Identify molecules and get their IUPAC names

Key Advantages

100% Free: No subscriptions, no hidden costs
No Installation: Runs entirely in your browser
Cross-Platform: Works on Windows, Mac, Linux, and mobile devices
Privacy-Focused: Your data never leaves your device

Getting Started

Getting started with MolDraw is incredibly simple. Here's how to begin:

  1. Visit the Website: Go to moldraw.scidart.com
  2. No Sign-Up Required: Just open the page and start using it immediately
  3. Familiarize Yourself: The interface is split into two panels:
    • Left Panel: 2D molecular structure editor (Ketcher)
    • Right Panel: 3D molecular viewer

Your First Molecule

Try drawing a simple molecule like benzene or water. Click on the atom tool in the left panel, then click on the canvas to place atoms. Connect them by clicking and dragging between atoms.

Drawing Molecular Structures

The left panel contains the Ketcher molecular editor. Here's how to draw structures:

Basic Drawing Tools

  • Atom Tool: Click to place atoms. Click on an existing atom to change its element
  • Bond Tool: Click and drag between atoms to create bonds
  • Selection Tool: Click and drag to select atoms/bonds for editing
  • Eraser Tool: Click on atoms or bonds to delete them

Advanced Features

  • Rings: Use the ring tool to quickly add common ring structures
  • Charges: Add formal charges to atoms
  • Stereochemistry: Indicate stereocenters with wedges and dashes
  • Text Labels: Add custom text annotations

3D Molecular Visualization

One of MolDraw's standout features is its integrated 3D viewer. As you draw or modify a structure, you can see it rendered in three dimensions in real-time.

Rendering Styles

  • Ball & Stick: Classic representation showing atoms as spheres and bonds as sticks (default)
  • Stick: Simplified view focusing on the molecular framework
  • Space-Fill: Van der Waals radii representation showing molecular volume
  • Line: Minimal wireframe view for clarity

Interactive Controls

  • Rotate: Click and drag to rotate the molecule
  • Zoom: Use mouse wheel or pinch gesture on touchscreens
  • Pan: Right-click and drag (or middle mouse button) to pan

Searching for Molecules

Don't want to draw from scratch? MolDraw integrates with PubChem to search for molecules by name!

How to Search

  1. Type a molecule name in the search bar at the top of the left panel
  2. Examples: "aspirin", "caffeine", "benzene", "glucose"
  3. Press Enter or click the search button
  4. MolDraw will fetch the structure from PubChem and load it into the editor

Exporting Your Structures

MolDraw supports multiple export formats to suit different needs:

Available Formats

  • PNG: High-quality image with transparent background - perfect for presentations
  • SDF: Structure Data Format - compatible with most chemistry software
  • XYZ: Coordinate format - ideal for computational chemistry
  • X3D: 3D model format with bonds - great for web visualization
  • OBJ: 3D model format - compatible with Blender and other 3D software

Advanced Tips & Tricks

Workflow Optimization

  • Start with Templates: Use ring structures and templates for common functional groups
  • Use Search First: If you know the molecule name, search for it rather than drawing from scratch
  • Edit After Search: Load a molecule from PubChem, then modify it to create derivatives

3D Visualization Tips

  • Switch Styles: Different rendering styles help visualize different aspects
  • Hide Hydrogens: For complex molecules, hiding hydrogens makes the structure clearer
  • Rotate Before Export: Find the best viewing angle before exporting PNG

Troubleshooting & FAQ

Common Issues

  • 3D View Not Updating: Make sure you've drawn a valid structure. Try refreshing the page if issues persist.
  • Search Not Working: Check your internet connection. PubChem requires an active connection.
  • Export Not Downloading: Check your browser's download settings.

Browser Compatibility

MolDraw works best on modern browsers (Chrome, Firefox, Safari, Edge). Make sure JavaScript is enabled.
