What is MolDraw?

MolDraw is a powerful, web-based molecular structure editor that allows you to:

  • Draw and edit chemical structures in 2D
  • Visualize molecules in 3D in real-time
  • Search for molecules by name using PubChem
  • Export structures in multiple formats (PNG, SDF, XYZ, X3D, OBJ)
  • Identify molecules and get their IUPAC names

Key Advantages

100% Free: No subscriptions, no hidden costs
No Installation: Runs entirely in your browser
Cross-Platform: Works on Windows, Mac, Linux, and mobile devices
Privacy-Focused: Your data never leaves your device

Getting Started

Getting started with MolDraw is incredibly simple. Here's how to begin:

  1. Visit the Website: Go to moldraw.scidart.com
  2. No Sign-Up Required: Just open the page and start using it immediately
  3. Familiarize Yourself: The interface is split into two panels:
    • Left Panel: 2D molecular structure editor (Ketcher)
    • Right Panel: 3D molecular viewer

Your First Molecule

Try drawing a simple molecule like benzene or water. Click on the atom tool in the left panel, then click on the canvas to place atoms. Connect them by clicking and dragging between atoms.

Drawing Molecular Structures

The left panel contains the Ketcher molecular editor. Here's how to draw structures:

Basic Drawing Tools

  • Atom Tool: Click to place atoms. Click on an existing atom to change its element
  • Bond Tool: Click and drag between atoms to create bonds
  • Selection Tool: Click and drag to select atoms/bonds for editing
  • Eraser Tool: Click on atoms or bonds to delete them

Advanced Features

  • Rings: Use the ring tool to quickly add common ring structures
  • Charges: Add formal charges to atoms
  • Stereochemistry: Indicate stereocenters with wedges and dashes
  • Text Labels: Add custom text annotations

3D Molecular Visualization

One of MolDraw's standout features is its integrated 3D viewer. As you draw or modify a structure, you can see it rendered in three dimensions in real-time.

Rendering Styles

  • Ball & Stick: Classic representation showing atoms as spheres and bonds as sticks (default)
  • Stick: Simplified view focusing on the molecular framework
  • Space-Fill: Van der Waals radii representation showing molecular volume
  • Line: Minimal wireframe view for clarity

Interactive Controls

  • Rotate: Click and drag to rotate the molecule
  • Zoom: Use mouse wheel or pinch gesture on touchscreens
  • Pan: Right-click and drag (or middle mouse button) to pan

Searching for Molecules

Don't want to draw from scratch? MolDraw integrates with PubChem to search for molecules by name!

How to Search

  1. Type a molecule name in the search bar at the top of the left panel
  2. Examples: "aspirin", "caffeine", "benzene", "glucose"
  3. Press Enter or click the search button
  4. MolDraw will fetch the structure from PubChem and load it into the editor

Exporting Your Structures

MolDraw supports multiple export formats to suit different needs:

Available Formats

  • PNG: High-quality image with transparent background - perfect for presentations
  • SDF: Structure Data Format - compatible with most chemistry software
  • XYZ: Coordinate format - ideal for computational chemistry
  • X3D: 3D model format with bonds - great for web visualization
  • OBJ: 3D model format - compatible with Blender and other 3D software

Advanced Tips & Tricks

Workflow Optimization

  • Start with Templates: Use ring structures and templates for common functional groups
  • Use Search First: If you know the molecule name, search for it rather than drawing from scratch
  • Edit After Search: Load a molecule from PubChem, then modify it to create derivatives

3D Visualization Tips

  • Switch Styles: Different rendering styles help visualize different aspects
  • Hide Hydrogens: For complex molecules, hiding hydrogens makes the structure clearer
  • Rotate Before Export: Find the best viewing angle before exporting PNG

Troubleshooting & FAQ

Common Issues

  • 3D View Not Updating: Make sure you've drawn a valid structure. Try refreshing the page if issues persist.
  • Search Not Working: Check your internet connection. PubChem requires an active connection.
  • Export Not Downloading: Check your browser's download settings.

Browser Compatibility

MolDraw works best on modern browsers (Chrome, Firefox, Safari, Edge). Make sure JavaScript is enabled.

Ketcher Overview

Ketcher is a powerful tool for drawing molecular structures and chemical reactions. It contains two modes:

  • Molecules mode: Allows drawing and operations with small/organic molecules and reactions
  • Macromolecules mode: Allows drawing and operations with biomolecules (nucleic acids and peptides)

Tool Palette

The tool palette provides access to all drawing and editing tools:

  • Draw and edit molecules by clicking and dragging atoms, bonds, and other elements
  • Delete elements using the Erase tool
  • Select fragments using Lasso, Rectangle, or Fragment selection tools
  • Access templates, rings, and functional groups

Ketcher Molecules Mode

Ketcher molecules mode consists of the following elements:

Note: Depending on the screen size, some tools on the Tool palette can be displayed in expanded or collapsed forms.

Using the Tool Palette

  • Draw and edit a molecule or a reaction by clicking on and dragging atoms, bonds, and other elements provided with the buttons on the Atoms toolbar and Tool palette
  • Delete any element of the drawing (atom or bond) by clicking on it with the Erase tool
  • Delete the entire molecule or its fragment using the Lasso, Rectangle, or Fragment selection tool and the Erase tool
  • Draw special structures (see the following sections)

Selection Methods

  • One atom or bond: Click Lasso or Rectangle Selection tool, then click the atom or bond
  • Entire structure: Use Fragment Selection tool and click the object, or use Lasso/Rectangle and drag, or Ctrl-click
  • Multiple elements: Shift-click to select multiple atoms/bonds, or drag around elements to select them
  • Multiple structures: Ctrl+Shift-click each structure

Main Toolbar Features

You can use the buttons of the Main toolbar:

File Operations

  • Clear Canvas: Start fresh
  • Open: Import molecules from files
  • Save As: Export in various formats
  • Copy/Paste/Cut: Standard clipboard operations

Editing Tools

  • Undo/Redo: Manage actions (Ctrl+Z / Ctrl+Shift+Z)
  • Aromatize/Dearomatize: Toggle aromatic representation
  • Layout: Auto-arrange structure
  • Clean Up: Improve appearance

Advanced Features

  • Calculate CIP: Determine R/S, E/Z stereoconfigurations
  • Check Structure: Validate molecular properties
  • Calculated Values: Display molecular properties
  • Add/Remove Hydrogens: Toggle explicit hydrogen display

Drawing Atoms

To draw/edit atoms you can:

  • Select an atom in the Atoms toolbar and click inside the drawing area
  • If the desired atom is absent from the toolbar, click on the button to invoke the Periodic Table and click on the desired atom (available options: Single – selection of a single atom, List – choose an atom from the list of selected options, Not List - exclude any atom on your list at that position)
  • Add an atom to the existing molecule by selecting an atom in the Atoms toolbar, clicking on an atom in the molecule, and dragging the cursor; the atom will be added with a single bond; vacant valences will be filled with the corresponding number of hydrogen atoms
  • Change an atom by selecting an atom in the Atoms toolbar and clicking on the atom to be changed; in the case a wrong valence appears the atom will be underlined in red
  • Change an atom by clicking on an existing atom with the Selection tool and starting to enter text after that
  • Change the charge of an atom by selecting the Charge Plus or Charge Minus tool and clicking consecutively on an atom to increase/decrease its charge
  • Change an atom or its properties by double-clicking on the atom to invoke the Atom Properties dialog (the dialog also provides atom query features)
  • Click on the button to use the Extended table and select a corresponding Generic group or Special Node

Drawing Bonds

To draw/edit bonds you can:

  • Click an arrow on the Bond tool in the Tool palette to open the drop-down list with the following bond types
  • Select a bond type from the drop down list and click inside the drawing area; a bond of the selected type will be drawn
  • Click on an atom in the molecule; a bond of the selected type will be added to the atom at the angle of 120 degrees
  • Add a bond to the existing molecule by clicking on an atom in the molecule and dragging the cursor; in this case you can set the angle manually
  • Change the bond type by clicking on it
  • Use the Chain Tool to draw consecutive single bonds
  • Change a bond or its properties by double-clicking on the bond to invoke the Bond Properties dialog
  • Clicking on the drawn stereo and dative bonds change their direction
  • Clicking with the Single Bond tool or Chain tool switches the bond type cyclically: Single-Double-Triple-Single

Drawing R-Groups (Markush Structures)

R-Groups allow you to create Markush structures representing families of compounds:

Creating R-Groups

  1. Use the R-Group Label Tool and click on an atom
  2. Select the R-group label (R1, R2, etc.) from the dialog
  3. Click Apply to convert the atom to an R-group

R-Group Members

  • Draw the structure you want as an R-group member
  • Use the R-Group Fragment Tool to select it
  • Assign it to the appropriate R-group label

Attachment Points

Use the Attachment Point Tool to mark which atom in the R-group member connects to the main structure. You can specify primary and secondary attachment points for multiple connections.

Marking S-Groups

S-Groups (Substructure Groups) allow you to mark and annotate parts of structures:

S-Group Types

  • Data S-Group: Attach data (names, values) to atoms or fragments
  • Multiple Group: Indicate number of replications in contracted form
  • SRU Polymer: Structural Repeating Unit for polymers
  • Superatom: Abbreviate part of structure to a text label
  • Query Component: Support SMARTS query properties
  • Nucleotide Component: Special superatom for nucleic acids
Superatom S-Groups: These allow you to abbreviate complex structures (like functional groups) to simple text labels, making drawings cleaner and more readable.

Drawing Reactions

Ketcher supports drawing chemical reactions with reagents, products, and arrows:

Creating Reactions

  1. Draw reagents and products as separate structures
  2. Use the Reaction Arrow Tool to add arrows between them
  3. Add plus signs using the Reaction Plus Tool for multiple reagents/products

Reaction Mapping

  • Auto-Mapping: Automatically maps atoms between reagents and products
  • Manual Mapping: Manually assign mapping numbers
  • Unmapping: Remove mapping numbers

Pathway Reactions

Use the Multi-Tailed Arrow Tool to create pathway reactions with multiple reagents or products. Adjust arrow length and position by dragging the ends.

Templates & Functional Groups

Templates provide quick access to common ring structures and functional groups:

Ring Templates

  • Benzene, Cyclopentadiene, Cyclohexane
  • Cyclopentane, Cyclopropane, Cyclobutane
  • Cycloheptane, Cyclooctane

Using Templates

  • Select template and click to add to canvas
  • Click on existing atom to fuse template
  • Drag from atom to add with single bond connection
  • Click on bond for bond-to-bond fusion

Functional Groups

Access functional groups from the Templates library. Functional groups can be added to structures and can be expanded, contracted, or have abbreviations removed as needed.

Custom Templates: Draw a structure and save it as a custom template for reuse. Access the Template Library via the Custom Templates button.

Stereochemistry

Ketcher supports comprehensive stereochemistry representation:

Stereocenters

  • Use wedged and dashed bonds to indicate stereochemistry
  • Calculate CIP: Button automatically assigns R/S, E/Z labels
  • Supports tetrahedral (R/S, r/s) and double bond (E/Z) stereochemistry

Enhanced Stereochemistry

  • Assign Absolute, AND, OR, or Mixed stereo flags
  • Configure stereo display in Settings
  • Control stereo label appearance and colors

Bond Stereochemistry

Single bonds with up/down wedges indicate tetrahedral stereocenters. Double bonds can show cis/trans (E/Z) stereochemistry. Click on stereo bonds to change their direction.

Working with Files

Ketcher supports numerous file formats for import and export:

Molecules Mode Formats

  • Ket Format
  • MDL Molfile (V2000, V3000)
  • SDF, RDF
  • SMILES (Daylight, Extended)
  • CML, InChI, InChIKey
  • SVG, PNG
  • CDXML, CDX

Macromolecules Mode

  • Ket Format
  • MDL Molfile V3000
  • Sequence (1-letter, 3-letter)
  • FASTA
  • HELM
  • IDT, AxoLabs formats
  • SVG

Opening Files

  • Open from File: Browse and select files from your computer
  • Paste from Clipboard: Paste file contents directly
  • Open as New: Clears canvas and loads structure
  • Add to Canvas: Adds structure without clearing existing work

3D Viewer

The 3D Viewer button on the toolbar opens a modal window displaying the structure in three dimensions:

3D Viewer Controls

  • Rotate: Hold the left mouse button and drag to rotate the structure
  • Zoom: Use mouse wheel to zoom in/out

3D Viewer Settings

Ketcher Settings allow changing the appearance of the structure and background coloring on the 3D Viewer tab. Default settings include:

  • "Lines" drawing method
  • "Bright" atom name coloring method
  • "Light" background coloring

Changing Structure Display

Rotate Tool

The Rotate tool allows you to change the orientation of structures:

  • If objects are selected, rotates all selected objects
  • If no objects are selected, rotates the whole canvas
  • Default rotation step is 15 degrees (changeable in Settings)
  • Hold Ctrl for continuous rotation with 1 degree step
  • Select a bond and press Alt+H to place it horizontally
  • Select a bond and press Alt+V to place it vertically

Flip Tool

  • Horizontal Flip (Alt+H): Flips selected objects or entire structures horizontally
  • Vertical Flip (Alt+V): Flips selected objects or entire structures vertically

Highlighting Atoms and Bonds

Right-click on an atom, bond, or multiple selected elements to highlight them with one of eight available colors.

ACS (American Chemical Society) Style

From Settings, you can apply ACS style settings. Structures will change their appearance to comply with ACS publication standards. This can be reset to default values.

Drawing Graphical Objects

Ketcher allows you to add graphical elements to your drawings:

Shape Tools

Click the arrow on the Shape Ellipse tool to access:

  • Shape Ellipse: Draw ellipses and circles
  • Shape Rectangle: Draw rectangles and squares
  • Shape Line: Draw straight lines

Adding Images

  • Select the Add Image tool from the Tools palette
  • Click anywhere on the canvas
  • Choose an image file (.png or .svg)
  • The image center will be placed at the click location

Adding Text Objects

  • Click the Add Text tool (Alt+T)
  • Click on the canvas to open the Text editor
  • Type text and apply formatting (bold, italic, subscript, superscript)
  • Double-click text objects to edit them
  • Add special symbols using the omega button

Ketcher Macromolecules Mode

Macromolecules mode allows drawing and operations with biomolecules (nucleic acids and peptides).

Main Toolbar Features

  • Clear Canvas: Clear the drawing area
  • Open/Save As: Import/export macromolecule files
  • Calculate Properties: Display molecular properties
  • Create Antisense Strand: Generate DNA/RNA antisense strands
  • Sync/Non-sync switcher: Control editing style for double-stranded sequences
  • Mode switcher: Switch between sequence, snake, and flex layout modes
  • Sequence mode typing type: Change interpretation of keyboard input (DNA/RNA/Peptide)

Layout Modes

Macromolecules mode offers three layout modes:

  • Sequence Mode: Text-editor-like view with single letter codes
  • Snake Mode: Visual representation with snake-like bonds
  • Flex Mode: Visual representation with straight bonds

Macromolecules Library

The macromolecules library has 4 tabs for organizing monomers and presets:

Library Tabs

  • Favorites: User-selected favorite monomers (starred items)
  • Peptides: Amino acid monomers organized alphabetically by one-letter codes
  • RNA: RNA builder, presets, and monomers (sugars, bases, phosphates, nucleotides)
  • CHEM: Non-RNA and non-amino acid monomers (linkers, tags)

Searching the Library

Search using monomer names, symbols, or IDT aliases. Hover over library cards to see detailed previews including:

  • Monomer name and chemical structure
  • Attachment point positions
  • HELM aliases
  • IDT and AxoLabs aliases

RNA Builder

The RNA Builder creates presets (collections of 2-3 monomers) for nucleotides, nucleosides, or sugar-phosphate pairs. Presets can be saved and reused.

Sequence Mode

Sequence layout mode provides a text-editor-like view for macromolecules. All nucleotides and amino acids are represented with single letter codes.

Visual Indicators

  • Nucleic Acids: Gray background = modified base, black frame = modified sugar, dot = modified phosphate
  • Peptides: Modified amino acids are underlined
  • Other Elements: Represented with @ symbol

Working in Sequence Mode

  • Type or paste text to add monomers (based on typing type switcher)
  • Backbone connections (R1-R2) are established automatically
  • Click library cards to add modified monomers
  • Select and click library cards to replace monomers
  • Right-click to modify nucleotides in RNA Builder or amino acids with modifications

Typing Type Switcher

Change how keyboard input is interpreted: DNA nucleotide, RNA nucleotide, or amino acid. Available only in sequence layout mode.

Snake and Flex Modes

Unlike sequence mode, snake and flex modes represent every monomer with a shape:

Monomer Shapes

  • Hexagons: Amino acids (color-coded by type: red=acidic, blue=basic, purple=polar, yellow/green=non-polar)
  • White boxes: CHEMs
  • Blue rounded squares: Sugars
  • Rhombuses: Bases (gray=purines, yellow/orange/purple=pyrimidines)
  • Coral circles: Phosphates
  • Pentagons: Unsplit nucleotides
  • Dark rounded squares: Unresolved IDT/AxoLabs nucleotides

Differences Between Snake and Flex

  • Bond Appearance: Flex uses straight lines; snake uses "snake-like" bonds with joints
  • Auto-layout: Snake mode automatically layouts structures; flex mode does not

Adding Monomers

Monomers can be added by: opening from file, pasting, drag-and-drop from library, or using the arrow icon on library cards.

Creating Antisense Chains

Create DNA or RNA antisense strands for selected sequences:

Creating Antisense Strands

  • Select the desired sense chain
  • Use the Create Antisense Strand icon or right-click menu
  • Choose Create Antisense DNA Strand or Create Antisense RNA Strand
  • The antisense chain is added below with hydrogen bonds established

Working with Double-Stranded Sequences

In sequence layout mode, a minus/line symbol represents bonds connecting monomers on both sides, allowing representation of non-linear structures like bulge loops.

Sync/Non-sync Editing

  • Sync On: Changes on one side impact the other side (complementary base pairing)
  • Sync Off: Changes on one side do not affect the other side chemistry

Hydrogen Bonds

Establish or delete hydrogen bonds via the right-click menu. When deleting, all hydrogen bonds involving selected monomers are removed.

Macromolecules Properties

Click the Calculate Properties button (Alt+C) to view molecular properties:

General Properties

  • Molecular Formula: Available for any structure
  • Molecular Mass: In Da, kDa, or MDa units

Peptide-Specific Properties

  • Isoelectric Point: Median of all pKa values
  • Extinction Coefficient: For λ=280nm (Gill & von Hippel method)
  • Hydrophobicity: Graph showing hydrophobicity score (Black & Mould method)
  • Amino Acid Count: Count of each amino acid by natural analogue

Nucleic Acid Properties

  • Melting Temperature: For double-stranded sequences (Khandelwal & Bhyravabhotla method)
  • Nucleotide Count: Count of each base

Using Properties

The Calculate Properties window is available in all modes. If multiple chains exist, select the chain you want to analyze.

Macromolecules and Molecules Mode Integration

Ketcher allows working with both small molecules and macromolecules:

Cross-Mode Visibility

  • Macromolecule structures are visible (but not editable) in molecules mode
  • Molecule structures are visible (but not editable) in macromolecules mode
  • Monomers appear differently in molecules mode (full abbreviations) vs macromolecules mode (shapes or codes)

Establishing Bonds Between Modes

  1. Draw the molecular structure in molecules mode
  2. Draw the macromolecular structure in macromolecules mode
  3. In macromolecules mode, use the Bond tool to connect from a monomer attachment point to a molecular atom

Expanding Monomers

In molecules mode, right-click on a monomer and select Expand monomer to view its full structure. Multiple monomers can be expanded simultaneously.

Expanded monomers behave like S-groups and highlight attachment atoms in blue and leaving groups in yellow when hovered.

Keyboard Shortcuts

Master these shortcuts to work more efficiently:

General (Ctrl = Mod)

  • Ctrl+Delete - Clear Canvas
  • Ctrl+O - Open
  • Ctrl+S - Save As
  • Ctrl+Z - Undo
  • Ctrl+Shift+Z - Redo
  • Ctrl+C - Copy
  • Ctrl+V - Paste
  • Ctrl+A - Select All

Tools

  • Escape - Cycle selection tools
  • Delete - Erase
  • 1, 2, 3, 4 - Bond types
  • Alt+R - Rotate Tool
  • Alt+H - Horizontal Flip
  • Alt+V - Vertical Flip
  • Alt+T - Add Text

Atoms

  • H, C, N, O - Common atoms
  • S, P, F, Cl - Other atoms
  • R - Pseudoatom
  • / - Atom Properties

Server Operations

  • Alt+A - Aromatize
  • Ctrl+Alt+A - Dearomatize
  • Ctrl+L - Layout
  • Alt+S - Check Structure
Note: On Mac, use Command instead of Ctrl. Press ? or Shift+/ in Ketcher to view all shortcuts.
