Frequently Asked Questions

Everything you need to know about MolDraw

Is MolDraw really free?
Yes, MolDraw is completely free to use. There are no hidden costs, subscriptions, or premium features. It's a free alternative to ChemDraw, built by Scidart Academy to make molecular editing accessible to everyone.
What file formats can I export?
MolDraw supports multiple export formats including:
  • PNG: High-quality images with transparent backgrounds
  • SDF: Structure Data Format for compatibility with chemistry software
  • XYZ: Coordinate format for computational chemistry
  • X3D: 3D model format with bonds for web visualization
  • OBJ: 3D model format compatible with Blender and other 3D software
Do I need to install anything?
No installation required. MolDraw runs entirely in your web browser. Just visit the website and start drawing molecules. No downloads, no account creation, no credit card required.
Can I search for molecules by name?
Yes! MolDraw integrates with PubChem to search molecules by name. Simply type a molecule name like "aspirin", "caffeine", "benzene", or any common compound name, and MolDraw will fetch the structure and load it into the editor.
What browsers are supported?
MolDraw works on all modern web browsers including Chrome, Firefox, Safari, Edge, and Opera. It requires JavaScript to be enabled. For the best experience, we recommend using the latest version of your browser.
Can I use MolDraw offline?
MolDraw requires an internet connection to function as it loads resources from CDNs. However, once loaded, many features work without constant connectivity. For full offline functionality, you would need to host MolDraw locally.
Is my data stored or shared?
No, MolDraw does not store your molecular structures. Everything runs locally in your browser. Your data never leaves your device, ensuring complete privacy and security.
How accurate is the 3D visualization?
MolDraw uses 3Dmol.js for accurate 3D molecular visualization. The structures are generated from 2D coordinates and provide realistic representations suitable for educational and research purposes. The visualization includes multiple rendering styles (ball & stick, stick, space-fill, line) to suit different needs.
Who built MolDraw?
MolDraw is built by Scidart Academy, created by Rafeeque Mavoor. It's powered by the open-source Ketcher chemical structure editor. For questions or feedback, contact rafeequemavoor@gmail.com.
Can I use MolDraw for commercial purposes?
Yes, MolDraw is free to use for both personal and commercial purposes. However, please note that MolDraw uses Ketcher (open-source) and 3Dmol.js, so you should review their respective licenses if you plan to use MolDraw in a commercial product.
